# serverless-security-monitoring

## Overview
This project demonstrates a real-time serverless security monitoring system built using AWS.

## Architecture
User → Lambda → EventBridge → Detection Engine → SNS → Email → DynamoDB

## Features
- Real-time threat detection
- Event-driven architecture
- Automated alerting
- Incident storage

## AWS Services Used
- AWS Lambda
- Amazon EventBridge
- Amazon SNS
- Amazon DynamoDB
- Amazon CloudWatch
